# Moony's Portfolio - Visual Web Showcase

このポートフォリオは、月の満ち欠けアニメーションを中心に構成されたインタラクティブなウェブポートフォリオです。  
音楽・映像・ビジュアル作品を一つの世界観に統合して表現しています。

## 🌕 概要

- 月の影アニメーションによるインパクトあるヒーローセクション
- Instagram・YouTubeへのリンクで作品に直接アクセス
- オリジナル音楽 & MVセクションあり
- スクロール連動でスムーズなユーザー体験
